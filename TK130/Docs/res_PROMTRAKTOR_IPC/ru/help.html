3D-каталог: интерфейс пользователя

Интерфейс 3D-каталога состоит из двух рабочих областей: 2D/3D-окно и DPL-таблица. Каждая рабочая область имеет свои элементы управления, описанные ниже.

 
  • 3D-каталог может быть представлен в трех графических режимах просмотра:
      Режим 2D и 3D обеспечивают просмотр 2D- и 3D-графических изображений в 2D- и 3D-окнах соответственно. В этом режиме 3D-каталог имеет кнопку ( 2D-графика) для переключения между окнами.
      Режим 3D позволяют использовать только 3D-иллюстрации.
      Режим 2D позволяют использовать только 2D-иллюстрации. В этом режиме доступны только листы, содержащие 2D-иллюстрации. Другие листы каталога недоступны.
    Режим просмотра задается при создании каталога и не может быть изменен во время просмотра.
  • Цветовая схема, описанная в справке, представляет цвета, установленные по умолчанию.

2D/3D окно

3D-окно предоставляет трехмерную иллюстрацию текущего листа каталога. При помощи команд навигации имеется возможность просматривать трехмерную иллюстрацию. Направляющие оси в правом верхнем углу 3D-окна предоставляют возможность пользователю определить свое положение и направление вида в трехмерном пространстве.

2D-окно (Arbortext IsoView) показывает изображения в формате CGM, соответствующие листу каталога.

Имя Кнопка Назначение
Лист   Позволяет выбрать соответствующий лист каталога из списка.
Восстановить Восстанавливает первоначальный (установленный автором) вид текущего листа.
2D-графика Осуществляет переключение между 2D- и 3D-окнами. Включает/выключает режим 2D-графики. Указанная кнопка появляется только в графическом режиме 2D и 3D.
Печать графики Печать содержимого 2D- или 3D-окна.
Выноски (только для 3D-окна) Включает/выключает режим нанесения выносок. Данный режим позволяет наносить выноски поверх изображения 3D-окна. Если режим нанесения выносок включен, навигационная панель замещается панелью выносок.
Игнорировать прозрачность (только для 3D-окна) Если включено, позволяет активировать чувствительные поля (щелчок кнопки мыши) через прозрачные геометрические поверхности в 3D-окне.
Режим анимации (только для 3D-окна) Если включено, постепенный переход (анимация) при изменении листа не используется.
Показать только графику Скрывает DPL-таблицу и полностью разворачивает 3D или 2D окно.

Навигационная панель 2D

Содержит кнопки навигационных команд в 2D-окне.

Имя Кнопка Назначение
Pan Позволяет перемещать изображение на в окне.
Zoom in Позволяет увеличить изображение.
Zoom out Позволяет уменьшить изображение.
Home Восстанавливает первоначальный вид изображения.

Панель навигации

Содержит кнопки навигационных команд в 3D-окне.

Имя Кнопка Назначение
Вращение Позволяет осуществлять вращение относительно определенного автором документа центра в трехмерном пространстве.
Перемещение Позволяет осуществлять перемещение точки наблюдения параллельно плоскости экрана.
Контроль увеличения Позволяет приблизиться или удалиться относительно трехмерного объекта.
Установить центр Позволяет установить центр вращения в трехмерном пространстве.
Показать/Скрыть центр вращения Позволяет показать или скрыть центр вращения в трехмерном пространстве.
Подогнать Устанавливает вид таким образом, чтобы 3D-графика была полностью видна в 3D-окне.

Представленные в таблице команды позволяют показывать в 3D-окне графические объекты с различных направлений:

Имя Кнопка Назначение
Вид спереди Показывает графический объект спереди.
Вид сзади Показывает графический объект сзади.
Вид слева Показывает графический объект слева.
Вид справа Показывает графический объект справа.
Вид свеху Показывает графический объект сверху.
Вид снизу Показывает графический объект снизу.
Изометрический вид Показывает изометрический вид графического объекта.

Панель выносок (в 3D-окне)

Замещает навигационную панель в режиме нанесения выносок (кнопка Выноски нажата). Предоставляет наложение выносок поверх изображения в 3D-окне. Полученное изображение может быть распечатано на принтере. В режиме нанесения выносок команды навигации недоступны.

Имя Назначение
Текст выноски Текстовое поле для ввода текста выноски.
Отменить Удаляет последнюю добавленную выноску.

Панель Объект (для 3D-окна)

Предоставляет команды настройки на вид, показать объект, скрыть объект, а также контроль прозрачности.

Имя Назначение
Пдгн (Подогнать выделенное) Подгоняет вид объекта таким образом, чтобы объект был показан достаточно близко и целиком умещался в 3D-окне.
Пкзт (Показать выделенное) Показывает объект, который был предварительно скрыт в 3D-окне.
Скрт (Скрыть выделенное) Скрывает объект в 3D-окне.
Прозрачность Позволяет установить прозрачность для выделенного объекта.
DPL-таблица

Содержит DPL-таблицу.

Имя Кнопка Назначение
Печать таблицы Печать DPL-таблицы.
Маркеры   Показывает или скрывает колонку маркеров в DPL-таблице. Колонка маркеров позволяет скрывать объекты в 3D-окне.
  • черный флажок – данной строке соответствует объект (деталь или сборка) в 3D-окне для текущего листа; все графические компоненты объекта представлены на данном листе.
  • серый флажок – данной строке соответствует объект (деталь или сборка) в 3D-окне для текущего листа; не все графические компоненты объекта представлены на данном листе.
  • флажок снят – объект (деталь или сборка) скрыт в 3D-окне.
    • перечеркнутый – объект (деталь или сборка) не имеет графического представления.
  • маркер отсутствует – объект (деталь или сборка) не имеет графического представления на текущем листе.
Вся таблица   Показывает всю DPL-таблицу. Серым отображаются строки, которые не соответствуют текущему листу.
Экспорт в Excel Сохраняет выделенные строки таблицы в формате Microsoft Excel. Если не выделено ни одной строки, сохраняются все строки текущей таблицы.
Справка Показывает текст данной справки.
Область заголовка   Область заголовка содержит имя проекта и дату опубликования.
DPL-таблица: основные действия пользователя
Операция Действие пользователя Результат
Подсветить строку Переместите указатель на требуемую DPL-строку. Если объект, соответствующий DPL-строке, проиллюстрирован в 2D или 3D окне, строка подсвечивается серым. Соответствующий объект в 3D-окне также подсвечивается.
 
  • Если объект, соответствующий DPL-строке, не проиллюстрирован в 2D и 3D окне, строка подсвечивается розовым.
Выделить объект Щелкните соответствующую DPL-строку. DPL-строка и соответствующие 3D/2D-графические объекты подсвечиваются желтым.
Выделить несколько объектов Удерживайте нажатой клавишу CTRL, щелкните требуемые DPL-строки. Требуемые DPL-строки последовательно выделяются.
Выделить группу строк, расположенных одна за другой Щелкните первую DPL-строку, нажмите клавишу SHIFT и затем щелкните последнюю DPL-строку. Группа строк выделяется.
Настроить вид на объект Дважды щелкните требуемую DPL-строку. Камера в 3D-окне устанавливается так, чтобы габарит выбранного объекта полностью помещался в 3D-окне.
Показать объект в 3D-окне Убедитесь, что маркеры включены, установите флажок на требуемой строке. Соответствующий объект будет показан в 3D-окне.
Скрыть объект в 3D-окне Убедитесь, что маркеры включены, снимите флажок на требуемой строке. Соответствующий объект будет скрыт в 3D-окне.
2D-окно: основные действия пользователя

Выноски в 2D-окне представляют гиперссылки, указывающие на соответствующие строки DPL-таблицы.

Операция Действие пользователя Результат
Подсветка Переместите указатель на требуемую выноску. Выноска и соответствующая DPL-строка подсвечиваются. Также показывается номер позиции.
Выделение Щелкните требуемую выноску. Выноска и соответствующая DPL-строка подсвечиваются желтым.
Перемещение (навигация) Щелкните кнопку Pan на навигационной панели, переместите указатель в 2D-окно и двигайте мышь, удерживая кнопку мыши. Изображение в 2D-окне перемещается вверх, вниз, влево, вправо.
Увеличение (навигация) Щелкните кнопку Zoom In на навигационной панели, а затем щелкните 2D-изображение.
 
  • В режиме увеличения также имеется возможность уменьшить изображение. Для этого, щелкните изображение, удерживая клавишу SHIFT на клавиатуре.
 Изображение увеличивается.
Уменьшение (навигация) Щелкните кнопку Zoom на навигационной панели, а затем щелкните 2D-изображение.
 
  • В режиме уменьшения также имеется возможность увеличить изображение. Для этого, щелкните изображение, удерживая клавишу SHIFT на клавиатуре.
 Изображение уменьшается.
3D-окно: основные действия пользователя
Операция Действие пользователя Результат
Подсветка Переместите указатель на требуемый 3D-объект. 3D-объект и соответствующая DPL-строка подсвечиваются. Также показывается номер позиции.
Выделение Щелкните требуемый 3D-объект. 3D-объект и соответствующая DPL-строка подсвечиваются желтым.
Выделение нескольких объектов Удерживайте нажатой клавишу CTRL, щелкните требуемые 3D-объекты. 3D-объекты выделяются последовательно.
Выделение группы 3D-объектов, расположенных в одной иерархической группе. Щелкните требуемый 3D-объект, затем нажмите и удерживайте клавишу SHIFT, повторяйте щелчки по 3D-объекту до тех пор, пока все необходимые объекты верхних уровней иерархии не будут выделены. Выделяется 3D-объект и объекты верхних уровней иерархии как в 3D-окне так и в DPL-таблице.
 
  • В 3D-окне выделяются объекты на всех указанных иерархических уровнях, в то время как в DPL-таблице выделяется только верхний по иерархии уровень.
Контроль увеличения (навигация)
  • Вращайте колесо мыши вперед или назад.

    • – или –
  • Щелкните "Контроль увеличения" на навигационной панели, затем переместите указатель в 3D-окно, нажмите левую кнопку мыши, перемещайте мышь вперед или назад, не отпуская левую кнопку.

    • – или –
  • Перетащите движок Контроль увеличения влево или вправо.
Объект приближается или удаляется в 3D-окне.
Перемещение (навигация)
  • Переместите указатель в любое место 3D-окна, затем перемещайте мышь, удерживая среднюю кнопку (или колесо мыши) в нажатом положении.

    • – или –
  • Щелкните Перемещение на навигационной панели, переместите указатель в 3D-окно и перемещайте мышь, удерживая левую кнопку.
Объекты в 3D-окне перемещаются вверх, вниз, влево, вправо.
Вращение (навигация) Щелкните Вращение на навигационной панели, переместите указатель в 3D-окне и перемещайте мышь, удерживая левую кнопку. Камера в 3D-окне двигается по окружности с центром, определенным автором трехмерной иллюстрации.
Установить центр вращения (навигация)
  • Нажмите клавишу ALT на клавиатуре и затем щелкните требуемый трехмерный объект в 3D-окне.

    • – или –
  • Щелкните Установить центр на навигационной панели и затем щелкните требуемый трехмерный объект в 3D-окне.

    • – или –
  • Поместите указатель на требуемый объект в 3D-окне и затем щелкните средней (или колесом) кнопкой мыши.
 Центр вращения устанавливается на поверхности трехмерного объекта.
Настройка вида на видимые объекты Убедитесь, что флажки маркера, соответствующие требуемым объектам, установлены в DPL-строке. Щелкните Подогнать на навигационной панели. Вид в 3D-окне устанавливается таким образом, чтобы 3D-графика отмеченных в таблице объектов была полностью видна.
Настройка вида на выделенные объекты Выделите требуемые объекты в 3D-окне или DPL-таблице, затем щелкните Пдгн (Подогнать выделенное) на панели Объект. Вид в 3D-окне устанавливается таким образом, чтобы 3D-графика выделенных объектов была полностью видна.
Настройка вида на объект с различных направлений Щелкните требуемую кнопку (Спереди, Сзади, Слева, Справа, Сверху, Снизу, изометрический) на навигационной панели. Графический объект показывается в 3D-окне согласно выбранному направлению.
Показать Выделите DPL-строку, соответствующего скрытого 3D-объекта, щелкните Пкзт на панели Объект.
 
  • DPL-строка скрытого объекта не имеет установленного флажка в колонке Маркеры и при наведения указателя мыши подсвечивается розовым.
Скрытый 3D-объект появляется в 3D-окне.
Скрыть Выделите DPL-строку, соответствующую 3D-объекту, щелкните Скрт на панели Объект. 3D-объект скрывается в 3D-окне.
Установка уровня прозрачности 3D-объекта Выделите 3D-объект в 3D-окне или DPL-таблице, переместите движок Прозрачность на панели Объект для достижения требуемого уровня прозрачности. Уровень прозрачности объекта в 3D-окне изменяется.
Печать содержимого 3D-окна с выносками
  1. С помощью команд навигации установите требуемый вид в 3D-окне.
  2. Щелкните кнопку Выноски, чтобы включить режим нанесения выносок.
  3. Щелкните DPL-строку, чтобы вставить номер позиции в поле Текст выноски.
  4. Если необходимо, отредактируйте текст в поле Текст выноски.
  5. Нажмите левую кнопку мыши в 3D-окне в месте, куда должна указывать линия выноски, затем, удерживая кнопку мыши, переместите указатель к месту, где должен располагаться текст выноски, отпустите кнопку мыши.
  6. Повторите шаги 3-5 для нанесения других выносок.
  7. Выполните печать изображения 3D-окна с помощью кнопки Печать графики.
  8. Щелкните кнопку Выноски, чтобы выключить режим нанесения выносок.
 
  • По умолчанию, текст выноски содержит номер позиции, соответствующий выделенной DPL-строке.
  • Чтобы удалить последнюю добавленную выноску, щелкните кнопку Отменить.
  • Чтобы удалить все созданные выноски и при этом сохранить установленный вид в 3D-окне, щелкните дважды кнопку Выноски.
  • Чтобы выключить режим нанесения выносок, выполните одно из действий: щелкните кнопку Выноски; щелкните кнопку Восстановить; выберите другой лист. Пожалуйста, обратите внимание, что все созданные выноски удаляются при выключении режима нанесения выносок.
 Текущее изображение 3D-окна выводится на печать.
